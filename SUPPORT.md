# Support

Use GitHub Issues for bugs, install problems, terminal/input behavior, docs issues, proposals, and support questions. GitHub Discussions are intentionally not used for this project.

Support is best effort. There is no response-time guarantee, and the maintainer may close, retitle, or narrow issues to keep the project focused.

Before posting, redact secrets, API keys, private repository paths, private task output, local config values, and terminal logs that should not be public. Include the output of `weft version`, your OS, terminal app, shell, install method, reproduction steps, and relevant redacted config or logs when they help explain the issue.
